🎮 ゲーム概要
**「Y_DRPG」**は、ブラウザで手軽に遊べる3Dダンジョン探索RPGです。 
ウィザードリィライクな主観視点で迷宮を探索し、モンスターを倒して装備を集め、世界の異変を解決するために5つのダンジョン踏破を目指します。

📜 ストーリー
世界に異変が起きている。 すべての迷宮を踏破せよ。 平和を取り戻すために。

✨ 特徴
3Dダンジョン探索: 往年の名作を彷彿とさせるグリッド移動式の3D迷宮。

ハクスラ要素: ドロップする装備品にはランダムな追加効果（ステータスアップ、HPアップ等）が付与されます。最強の装備を求めて探索しよう。

5つのダンジョン: 地下迷宮、迷いの森、海底洞窟、古代神殿、天空の塔。それぞれ推奨レベルや属性、ギミック（流水、ワープ、落とし穴）が異なります。

バトル: 属性相性（火・水・地・風・光・闇）やバフ・デバフを駆使するターン制コマンドバトル。

オートマッピング: 歩いた場所が自動的に記録されるミニマップ機能を搭載。

セーブ機能: ブラウザのローカルストレージを使用したセーブ＆ロードに対応。

🕹️ 操作方法
PC（マウス・キーボード）とスマートフォン（タッチ操作）の両方に対応しています。

基本操作
移動・コマンド: 画面内のボタンをクリック／タップ

キーボード操作（PC用）:

↑ / W: 前進

↓ / S: 後退

← / A: 左を向く

→ / D: 右を向く

Enter: 調べる・決定

🏰 町の施設
冒険の拠点となる「始まりの町」には以下の施設があります。

宿屋: HPの回復。全滅時の復帰場所。

道具屋: 冒険に役立つアイテムや装備品の購入・売却。

神殿:

レベルアップ: 経験値が貯まったらここでステータスを割り振って成長させます。

転職: 勇者・戦士・魔法使い・僧侶の4つの職業へ転職可能（レベルは1に戻ります）。

外へ: ワールドマップへ移動し、ダンジョンを選択します。

⚔️ パーティと職業
3人パーティで冒険します。

アベル (勇者/Hero): 攻守のバランスが良い万能型。

メイ (魔法使い/Mage): 強力な攻撃魔法のエキスパート。

シーラ (僧侶/Priest): 回復と支援魔法の要。

🚀 起動方法
特別なインストールは不要です。

フォルダ内の index.html をモダンブラウザ（Chrome, Firefox, Safari, Edgeなど）で開いてください。

「冒険を始める」ボタンを押すとゲームがスタートします。

📂 ファイル構成
index.html: ゲームのメインファイル

game.js: ゲームロジック、データ、描画処理

style.css: スタイルシート（レスポンシブ対応）

*.png: キャラクター、モンスター、アイコン等の画像リソース

⚠️ 注意事項
セーブデータはブラウザの「ローカルストレージ」に保存されます。ブラウザのキャッシュをクリアするとセーブデータが消える場合がありますのでご注意ください。
